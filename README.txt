Description:
This is a Python terminal-based Battleship game. The user chooses a board size 
and number of ships. The program randomly places ships and the user has a limited 
number of shots to try and hit the ships. The game displays hits, misses, the 
board after each shot, and shows the final score.

How to Run My App (Python):
1. Install Python 3.11 or higher.
2. Open a terminal in the folder containing battleship.py
3. Run the command:
      python battleship.py

How to Run Using Docker:
1. Pull my Docker image:
      docker pull andresborbor/battleship-game:latest
2. Run the container:
      docker run -it andresborbor/battleship-game
